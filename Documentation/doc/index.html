base.distance_measure(route_shp)[source]¶

Calculates the distance between points along the route and calculates the cumulative distance.

route_shp: shapefile for the selected route; output of read_shape()

distance: list containing the distance between each point [meters] cum_distance: array of the total route distance at each point along the route (e.g. the first point will have a distance of 0, and the last point will equal the total route distance) [meters]

base.extract_point_df(route_shp)[source]¶

Extracts coordinates for all points along the route.

route_shp: shapefile for the selected route; output of read_shape()

linestring_route_df: dataframe containing the coordinates for every point along the route

base.gradient(route_shp, rasterfile)[source]¶

Calculates the elevation and road grade at each point along the route.

route_shp: shapefile for the selected route; output of read_shape() rasterfile: elevation data file (.tif)

elevation_meters: the elevation at each point along the route route_gradient: the road grade (e.g. slope) at each point along the route route_cum_distance: the total route distance at each point along the route [m] route_distance: the distance between each point [m]

base.make_lines(gdf, gradient, idx, geometry='geometry')[source]¶

Creates a line between each point; iterative function for make_multi_lines(), so it is not called directly.

gdf: dataframe of coordinates; output of extract_pts_df() gradient: the road grade (e.g. slope) at each point along the route; output of gradient() idx: index geometry: DEFAULT = ‘geometry’

df_line: dataframe containing the line segments

base.make_multi_lines(linestring_route_df, elevation_gradient)[source]¶

Creates a geo dataframe containing the road grade and linestring for each point along the route.

linestring_route_df: dataframe of coordinates; output of extract_pts_df() elevation_gradient: the road grade (e.g. slope) at each point along the route; output of gradient()

gdf_route: geodataframe with columns [‘gradient’, ‘geometry’]

base.profile_plot(elevation, elevation_gradient, route_cum_distance, route_num)[source]¶

Creates two plots. First shows elevation vs. distance. Second shows absolute grade vs. distance.

elevation: the elevation at each point along the route elevation_gradient: the road grade at each point along the route route_cum_distance: the total route distance at each point along the route [m] route_num: route number (integer)

plt: Elevation vs. distance and absolute grade vs. distance plots

base.read_shape(shapefile, route_num)[source]¶

Reads shapefile and selects desired route.

shapefile: route geospatial data (.shp file) route_num: route number (integer)

route_shp: shapefile for the selected route

base.route_map(gdf_route)[source]¶

Creates interactive map for the desired route.

gdf_route: geodataframe output from make_multi_lines()

route_map: interactive map that displays the desired route and road grade

base.route_metrics(elevation, elevation_gradient, route_cum_distance, distance, route_num)[source]¶

Estimates route difficulty based on four test metrics.

elevation: the elevation at each point along the route elevation_gradient: the road grade at each point along the route route_cum_distance: the total route distance at each point along the route [m] distance: the distance between each point [m] route_num: route number (integer)

display_metrics: string of metrics results metric_values: results of metrics calculations
